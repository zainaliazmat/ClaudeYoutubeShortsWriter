# Audio source policy (monetization-safe)

The channel is monetized, so every track/SFX must be safe for YouTube monetization and free of Content-ID claims. Pick in this priority order.

## Allowed sources (free)
1. **YouTube Audio Library** (YouTube Studio → Audio Library) — the ONLY library YouTube itself certifies copyright-safe and won't Content-ID-claim. **Default choice.** Most tracks need no attribution; a subset marked **"CC BY"** require crediting the artist — record the exact credit string when so.
2. **Pixabay Music** — free, no attribution required.
3. **Uppbeat** — free monthly download tier (limited); each free track has a required credit code — record it.

## Allowed sources (paid, only if the user has an active subscription)
- **Epidemic Sound** / **Artlist** — pre-cleared for monetization. Clearance persists for content published while subscribed. Keep the subscription active while videos are public.

## Hard rejects
- **CC-BY-NC** anything (non-commercial — breaks monetization).
- Unknown / unstated license.
- Re-upload channels of otherwise-free tracks — these get Content-ID-flagged. Always link the ORIGINAL source.
- "Free for non-monetized use only" tracks.

## Always record per pick
title · artist · source library · direct URL · license · attribution required? (+ exact credit string) · monetization-safe? (must be yes)

## Caveats
- "Royalty-free" ≠ "copyright-free." Stock platforms disclaim third-party IP.
- Download from the original source at RENDER time (this pipeline only records the choice + license — no downloads).
- Keep license receipts; keep paid subscriptions active while videos are public.
